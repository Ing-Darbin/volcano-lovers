Guía de Edición del Sitio Web

Esta guía te explicará cómo realizar cambios básicos en tu sitio web (como textos, precios o imágenes) sin necesidad de saber programación, utilizando la interfaz de GitHub.

Importante: Todos los cambios que hagas se reflejarán en el sitio web público unos minutos después de guardarlos.

1. Acceder a los Archivos

Para editar el sitio, primero debes ir a donde están guardados los archivos:

  1. Ve a tu repositorio en GitHub: https://github.com/Ing-Darbin/volcano-lovers
  2. Asegúrate de haber iniciado sesión con tu cuenta.

2. Editar Textos (index.html)

La mayoría del contenido (títulos, descripciones, precios) está en el archivo index.html.

Pasos para editar:

  1. En la lista de archivos, haz clic en index.html.
  2. Busca el icono de un lápiz ✏️ en la esquina superior derecha del archivo (dice "Edit this file"). Haz clic en él.
  3. Ahora verás el código. No te asustes, solo busca el texto que quieres cambiar.
    • Usa Ctrl + F (o Cmd + F en Mac) para buscar la frase exacta que quieres modificar.
  4. Cambia el texto que está en color blanco o negro (dependiendo de tu tema). ¡Cuidado! No borres las etiquetas que están entre < > (como <h1>, <p>, <div>). Solo cambia el texto que está dentro de ellas.
  5. Una vez hecho el cambio, baja hasta el final de la página a la sección "Commit changes".
  6. Escribe un mensaje corto describiendo qué cambiaste (ej: "Actualizar precio del tour Telica").
  7. Haz clic en el botón verde "Commit changes".

3. Cambiar Imágenes

Para cambiar una imagen, lo más fácil es subir una nueva imagen con el mismo nombre que la anterior, o subir una nueva y cambiar el nombre en el código.

Opción A: Reemplazar imagen existente (Más fácil)

  1. Mira en la carpeta images qué nombre tiene la imagen que quieres cambiar (ej: logo.jpg).
  2. Prepara tu nueva imagen en tu computadora y ponle exactamente el mismo nombre.
  3. En GitHub, entra a la carpeta images.
  4. Haz clic en "Add file" > "Upload files".
  5. Arrastra tu nueva imagen.
  6. Haz clic en "Commit changes". GitHub reemplazará la vieja por la nueva.

Opción B: Usar una imagen nueva

  1. Sube tu imagen a la carpeta images (ej: nuevo-volcan.jpg).
  2. Ve al archivo index.html y edítalo.
  3. Busca donde se usa la imagen anterior (busca src="images/...").
  4. Cambia el nombre por el de tu nueva imagen: src="images/nuevo-volcan.jpg".
  5. Guarda los cambios ("Commit changes").

4. ¿Cuándo se ven los cambios?

Después de hacer clic en "Commit changes", GitHub procesará los cambios automáticamente. Esto suele tomar entre 1 a 3 minutos.

Si no ves los cambios, intenta refrescar tu página web (a veces es necesario borrar la caché del navegador).
